# EventEaseBookingSystem
The application manages venues, events, and bookings for EventEase and will be deployed to Microsoft Azure with a SQL Server database. 

This project is developed as part of the CLDV7111 Cloud Development A Portfolio of Evidence (POE).

## Technologies Used
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Microsoft Azure Web App

## Features
- Manage Venues
- Manage Events
- Create and manage Bookings
- Prevent double bookings
